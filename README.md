# PasswordManagerCPP

A command-line password manager built in C++. This application allows users to securely organize account information by storing website names, usernames, and passwords in a simple, menu-driven interface. Account data is saved to a file so it persists between program sessions.

## Features

* Add new accounts
* View saved accounts
* Hide passwords by default
* Reveal passwords for a selected account
* Delete accounts
* Save account information to a file
* Automatically load saved accounts when the program starts

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* Standard Template Library (STL)
* Vectors
* File Input/Output (fstream)
* String Streams (sstream)
* Git & GitHub
